# Chat Task Extractor

一个本地运行的聊天记录任务提取工具。用 AI 从微信聊天记录中自动提取任务、工作安排和时间节点，告别手动翻记录的焦虑。和导师/领导的微信聊天里散落着大量任务和安排，手动翻阅整理既费时又怕遗漏。这个工具帮你一键提取，生成清晰的任务清单。

它支持：

- 📂 上传微信聊天记录文件（txt / csv / json）
	•	🤖 AI 自动识别任务、时间节点、截止日期
	•	📊 生成可编辑的任务表格（日期、说话人、摘要、任务、截止时间、状态）
	•	✏️ 支持手动修改任务内容和状态
	•	📥 一键导出为 CSV 文件

这个项目默认按Claude 兼容接口工作，也可以改成你自己的兼容接口地址。

## 文件说明

- `index.html`
  前端页面
- `server.py`
  本地代理服务，负责转发 API 请求和写入 macOS 提醒事项
- `start.command`
  macOS 一键启动脚本

## 运行环境

- macOS
- Python 3
- 浏览器（Safari / Chrome 均可）

如果你要用“导出到提醒事项”，macOS 可能会提示授权终端或 `osascript` 访问提醒事项。

## 启动方式

### 方式一：双击启动

直接双击：

```bash
start.command
```

### 方式二：命令行启动

在项目目录执行：

```bash
python3 server.py
```

启动后浏览器会打开：

```text
http://127.0.0.1:8765/
```

需要填写：

- `API Key`
- `API URL`
- `模型 ID`

默认配置：

- `API URL`
  `https://dragoncode.codes`
- `模型 ID`
  `claude-opus-4-6`

如果你使用的是 Dragon Code 的 Claude 兼容 Key，这套默认值可以直接用。

## 提取逻辑

- 会自动把大文本切成多个小段
- 每段单独请求模型
- 自动合并结果
- 自动去重
- 会把说话人匿名化为 `A / B / C`

## 导出说明

### 导出 CSV

点击页面里的 `导出 CSV` 即可。

### 导出到提醒事项

1. 勾选要导出的任务
2. 点击 `导出到提醒事项`
3. 输入提醒事项列表名称

程序会尝试根据 `deadline` 或 `date` 自动识别提醒时间。

当前支持部分常见时间表达，例如：

- `明天`
- `后天`
- `明晚`
- `下周一`
- `4.10`
- `2026年5月27日`
- `15:20`
- `下午`
- `晚上`

如果某条任务没有识别出时间，这条任务会被跳过，不会乱写入提醒事项。

## 常见问题

### 1. 为什么浏览器里直接打开 HTML 不行？

因为这个项目不只是静态网页，还依赖本地代理服务。

不要直接双击 `index.html`。

请用：

- `start.command`
- 或 `python3 server.py`

### 2. 为什么会出现 JSON 格式错误？

因为模型偶尔会返回“不完全合法”的 JSON，例如：

- 字符串内部引号没转义
- 带多余说明文字
- 少逗号

前端已经做了自动清洗和修复，但极端情况下仍可能失败。

### 3. 为什么大文件容易失败？

单次请求太大时，第三方兼容接口可能直接断开连接。

这个项目已经默认开启：

- 自动分段
- 重叠切片
- 节流请求

### 4. 为什么提醒事项导出失败？

常见原因：

- macOS 没给权限
- 当前任务没有可识别日期
- 提醒事项 AppleScript 属性在当前系统版本行为不同

这时可以看页面里的调试信息。
